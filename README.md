# Mini Message Board

A simple message board application built with **Node.js**, **Express**, and **EJS**. Users can view messages and create new messages via a form. This project demonstrates **server-side rendering**, **form handling**, and **templating with EJS**, with a modern and responsive UI.

---

## Features

- View all messages in a clean, card-style layout.
- Open individual messages to see details.
- Create new messages via a form.
- Responsive design for desktop, tablet, and mobile.
- Accessible design (AA-level contrast, focus outlines).
- Dark mode support based on system preferences.

> Note: Only the user who submits a message can edit or delete it if that feature is implemented. Otherwise, messages are read-only.
